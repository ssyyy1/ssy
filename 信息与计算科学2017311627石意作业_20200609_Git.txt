
1. Git 和 SVN 两款工具的主要区别是什么？

4. 对于远程仓库 pull 和 clone 命令的区别是什么？
pull：$git checkout localbranch
$git pull origin master
$git branch
master 
*localbranch

clone：$cd newfolder
$git clone git@github.com:whatever/something.git
$git branch
*master
remotebranch

5. 说明本地工作目录、暂存区和本地仓库的区别和作用分别是什么？
工作区：工作区就是你克隆项目到本地后，项目所在的文件夹目录。
暂存区：用于存储工作区中添加上来的变更（新增、修改、删除）的文件的地方。

本地仓库：用于存储本地工作区和暂存区提交上来的变更（新增、修改、删除）过的文件的地方。

远程仓库：当某一个人的开发工作完毕时，需要将自己开发的功能合并到主项目中去，但因为功能是多人开发，如果不能妥善保管好主项目中存储的代码及文件的话，将会存在丢失等情况出现，所以不能将主项目放到某一个人的本地电脑上，这时就需要有一个地方存储主项目，这个地方就是我们搭建在服务器上的git远程仓库，也就是在功能开始开发前，每个人要下载项目到本地的地方。
6. 说明如何进行版本的回退？
命令： reset（重置）将暂存区和本地工作目录中的内容完全回退到指定的提交版本
git reset –hard HEAD ^

8. 简述在 Gitee 平台上，分支、标记的作用。
分支：多个人可以同时进行操作，不会影响到master主分支，所有人都可以正常工作。
标记：可以链接到一个文件、一个网址，多人工作时，更加清晰。
